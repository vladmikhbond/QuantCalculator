Векторы
-------
<a|b> - скалярное произведение a† и b (комп число) a† - эрмитово сопряжение 
<a|b> = <b|a>*
√<a|a> - норма вектора
<a|b> = 0 => векторы ортогональны
aⱼ = <eⱼ|a> - компонента равна скал. произведению вектора на базисный вектор

|a>|b> - тензорное произведение векторов
∑ⱼ|aⱼ><aⱼ| = 1

Эрмитовы операторы
------------------
Оператор S эрмитов, если S† = S. 
S = [[r1,   a+bi],   - эрмитовы (самосопряженные) операторы задаются квадратной матрицей
     [a-bi, r2  ]]

S|a> = |b>  - оператор действует на кет-вектор
<a|S† = <b| - оператор действует на бра-вектор
(A B)† = B†A†

Определение собственных векторов оператора 
M|a> = m|a>   - M - оператор, m - собственное значение (число)

Все собственные векторы эрмитова оператора отогональны друг другу.
Поэтому собственные векторы эрмитова оператора образуют базис.


Постулаты КМ
============
1. Наблюдаемым величинам (коорд., импульс, спин,...) соответствуют эрмитовы операторы.

2. Результатом измерения может быть только одно из собственных значений оператора.

3. При измерении величины вектор состояния этой величины переходит в собственный вектор ее оператора, 
   соответствующий измеренному собственному значению  (постулат об измерении).

4. Правило Борна. Квадрат абсолютного значения амплитуды вероятности есть вероятность события, 
   описываемого этой амплитудой.
       Амплитуда вероятности = <a|b>
       p(a|b) = |<a|b>|² = <a|b>* <a|b> = <b|a><a|b>,   p - вероятность a при условии, что b произошло. 

5. Значения величин, операторы котрых не коммутируют, невозможно знать одновременно.

6. Принцип суперпозиции. Произвольный вектор состояния можно представить как взвеш. сумму базисных. 
       |ψ> = c1|e1> + c2|e2>   
       c1 - амп.вер. того, что ψ перейдет в e1,  c1 = <e1|ψ>, c2 = <e2|ψ>
       |c1|² + |c2|² = 1
